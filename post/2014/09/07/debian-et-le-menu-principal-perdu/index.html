Debian et le "Menu Principal" perdu

Date dim. 07 septembre 2014 Tags debian

Je mets ça là au cas où ...

Pour une raison qui m'échappe, je n'ai plus eu de menu "Menu Principal", accessible en principe depuis le menu "Activités" > "Applications"

Or sans ce dernier, pour rajouter des applications qui ne sont pas issues de package debian, au hasard, un éditeur particulier, c'est peine perdue, obliger d'en passer par la ligne de commandes.

Le Sésame vient du package debian "alacarte"

apt-get install alacarte

zavourez comme nom de package, c'est mnémotechnique à ... mourir... pas loin de nom à la ikéa :P

et donc revoilà le menu

On est fin prêt pour remettre les applications de son choix dans le menu et via les Extensions, on retrouve son application dans le menu en haut de votre desktop
