Jelix versions 1.1.6 1.0.12 et Bitbucket

Date Sat 27 March 2010 Tags ApacheMySqlPHP / Jelix

Oyé Oyé Braves Gens

Voici Quelques nouvelles de Jelix.

Nouvelles versions :
A commencer par 2 nouvelles versions de mises à jour de maintenance pour les branches 1.1 et 1.0

Où pour la version 1.1.6, une petite manipulation est nécessaire suite au support des champs binaires dans jDb et jDao

Nouveaux dépots :
Ensuite, il est à noter que les dépôts même de Jelix sont à présent sur

Dans son sillage, auront bougé des projets de la forge tels :

  1. jCommunity : l'indispensable module pour gérer son site communautaire
  2. HaveFnuBB : le forum 100% Jelix powered , vrai composite des modules de la communauté
  3. jBlog : un blog 100% Jelix powered :)
  4. "dépôt de miel" : un ensemble de modules jelix indépendants, comme le download manager, etc..
  5. Rarangi : Le Générateur de documentation PHP
  6. jButracker : Le système de gestion de tickets, wiki, sources

Voilà de quoi satisfaire la majorité d'entre nous ;-)
